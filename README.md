# My Projects Dashboard

Welcome to the **My Projects Dashboard**! This project is a centralized hub containing a collection of beautifully designed web applications. It serves as a portfolio of interactive, responsive, and visually appealing tools built with modern web technologies.

## 🚀 Applications Included

### 1. TaskMaster Pro (`/To_do_`)
A feature-rich, interactive To-Do list application designed to help you organize your work effortlessly.
- **Key Features:**
  - Add tasks with priority levels (Low, Medium, High) and due dates.
  - Track task statistics (Total, Completed, Pending).
  - Search functionality to quickly find specific tasks.
  - Filters to view All, Pending, or Completed tasks.
  - **Kids Mode:** A fun, interactive theme with decorative characters and celebrations.
  - Light and Dark theme toggle.

### 2. ClimaView (`/weather`)
A modern, responsive weather dashboard providing real-time weather information with stunning visual effects.
- **Key Features:**
  - Search for weather forecasts by city.
  - Displays current temperature, weather conditions, humidity, and wind speed.
  - Realistic weather background animations (e.g., sun, rain clouds).
  - Shows highest and lowest temperature extremes.
  - 5-Day forecast breakdown.
  - Clean UI with Phosphor Icons and a modern dark theme.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom styling, modern layouts (Flexbox/Grid), responsive design, and CSS animations.
- **JavaScript (Vanilla):** Dynamic interactions, DOM manipulation, state management, and API data fetching.
- **Icons:** FontAwesome (TaskMaster Pro) & Phosphor Icons (ClimaView).
- **Fonts:** Google Fonts (Poppins, Fredoka).

## 💻 How to Run
1. Clone or download the repository to your local machine.
2. Open the root `index.html` file in any modern web browser.
3. Click on the respective application cards on the dashboard to launch them.

## ✨ Design Philosophy
The applications in this dashboard prioritize **visual excellence** and **dynamic design**. By utilizing vibrant color palettes, smooth gradients, and engaging micro-animations, the user interface is crafted to feel premium, state-of-the-art, and responsive to user interactions.
